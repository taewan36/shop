# BootShop
 스프링부트, 부트스트랩을 사용하여 만든 간단한 온라인 서점입니다.
 
 도로명 주소 api를 통해 도로명 주소를 검색할 수 있고
 
 결제는 아이엠포트api를 통해서 하고
 단순 결제뿐만 아니라 실제 결제가 되었는지 api를 통해 상호 검증하는 것까지 구현하였습니다.
 
 Aws ec2 linux 서버로 http://3.37.137.168:8090/   에 게시돼있습니다.
 
 DB는 MariaDB를 사용합니다.
